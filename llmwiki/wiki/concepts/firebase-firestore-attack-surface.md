related: [[Firebase / Firestore]]

Bề mặt tấn công của Firebase/Firestore tập trung vào các lớp sau:

- Data stores: Firestore, Realtime Database, Cloud Storage
- Authentication: ID tokens, custom claims, anonymous/sign-in providers, App Check
- Server-side: Cloud Functions, Admin SDK
- Infrastructure: Hosting rewrites, CDN/caching, CORS

Đây là các khu vực cần map trước khi kiểm thử bảo mật Firebase.
