Các bypass techniques được nhắc đến:

- attribute lookup thay vì direct access
- class walk / MRO walk
- string constructor games
- filter/function aliasing
- implicit conversion
- whitespace/case variants
- token assembly bằng concatenation
- encoding layering
- operator precedence games
- null byte truncation
- Unicode normalization
- polyglot và chained evaluation
