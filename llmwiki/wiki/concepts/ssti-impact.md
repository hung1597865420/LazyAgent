Tác động của SSTI:

- remote code execution trên rendering host
- data exfiltration qua gadget chains
- cloud credential theft qua metadata service
- lateral movement vào internal services
- persistent backdoor
- build / supply-chain compromise

Tài liệu coi RCE là outcome mặc định của SSTI.
