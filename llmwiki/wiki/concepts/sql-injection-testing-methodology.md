---
title: SQL Injection Testing Methodology
type: concept
related: [[SQL Injection]]
---

Quy trình kiểm thử:

1. Identify query shape
2. Determine input influence
3. Confirm injection class
4. Choose quietest oracle
5. Establish extraction channel
6. Pivot to metadata
7. Target high-value tables

Mục tiêu là chứng minh control tại boundary giữa input và SQL.
