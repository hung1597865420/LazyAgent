---
title: BFLA High-Value Actions
type: concept
related: [[Broken Function Level Authorization (BFLA)]]
---

Các hành động có giá trị cao cần kiểm thử BFLA:

- Đổi role/permission, impersonation/sudo, invite/accept vào org
- Approve/void/refund/credit issuance, override giá/plan
- Export/report generation, data deletion, account suspension/reactivation
- Feature flag toggles, quota/grant adjustments, license/seat changes
- Security settings như reset 2FA, override email/phone verification
