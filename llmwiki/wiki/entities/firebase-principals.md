title: Firebase Principals
type: entity

Các principal được nhắc đến để kiểm thử:

- Unauthenticated
- Anonymous
- Basic user A
- Basic user B
- Staff/admin

Chúng được dùng để xây dựng ma trận quyền truy cập và so sánh kết quả giữa các vai trò.
