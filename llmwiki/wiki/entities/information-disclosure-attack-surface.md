Các bề mặt disclosure được nhắc đến:

- Errors and exception pages
- Debug/dev tooling
- DVCS/build artifacts and temp/backup files
- Configuration and secrets
- API schemas and introspection
- Client bundles and source maps
- Headers and response metadata
- Storage/export surfaces
- Observability/admin endpoints
- Directory listings and indexing

Đây là các nguồn lộ thông tin cần được audit.
