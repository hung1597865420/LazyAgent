Các trick filename/path được nhắc đến:

- Unicode homoglyphs
- trailing dots/spaces
- device names
- reserved characters
- null-byte truncation
- overlong paths
- case-insensitive collisions

Chúng có thể giúp bypass validator hoặc overwrite file khác.
