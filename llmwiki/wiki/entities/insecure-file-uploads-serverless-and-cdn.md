Các thành phần serverless/CDN được nhắc đến:

- direct-to-bucket uploads
- Lambda/Workers post-processing
- CDN caching of uploaded content
- cache keys
- headers

Đây là nơi security decisions có thể bị delegate sai hoặc cache sai nội dung upload.
