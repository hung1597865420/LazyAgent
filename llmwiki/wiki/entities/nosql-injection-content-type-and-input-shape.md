Các content-type/input shape được nhắc đến:

- `application/json`
- `application/x-www-form-urlencoded`
- bracket notation như `username[$ne]=x`
- raw JSON operator objects
- backend using Mongoose, native MongoDB driver, or REST ODM wrapper
