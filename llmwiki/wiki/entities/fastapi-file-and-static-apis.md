Các API xử lý file và static content trong FastAPI:

- `UploadFile`
- `File`
- `FileResponse`
- `StaticFiles`

Chúng liên quan đến upload, download, và mount static assets, cần kiểm tra traversal, storage root, và cách phục vụ file.
