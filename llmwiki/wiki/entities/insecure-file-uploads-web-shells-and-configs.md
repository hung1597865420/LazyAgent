Các payload/config được nhắc đến:

- PHP GIF polyglot
- `<?php echo 1; ?>`
- `.htaccess`
- `.user.ini`
- `AddType`
- `AddHandler`
- `auto_prepend_file`
- `auto_append_file`
- ASP/JSP equivalents
- IIS `web.config`

Đây là các file hoặc cấu hình có thể bật execution trên server.
