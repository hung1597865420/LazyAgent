---
title: HTTP Request Smuggling Response Artifacts
type: entity
related: [[HTTP Request Smuggling Validation]]
---

Các artifact dùng để xác minh impact:

- 200 response
- 403 response
- unexpected response
- timing delay 10–30 seconds
- unique marker string
- raw bytes of smuggled request

Chúng là bằng chứng cần thiết để xác nhận desync và capture.
