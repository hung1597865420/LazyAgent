Các shape variant được nhắc đến:

- arrays vs scalars
- nested JSON
- objects under unexpected keys
- dot paths như `profile.role`
- bracket paths như `profile[role]`
- `settings[roles][]`
- duplicate keys như `{"role":"user","role":"admin"}`
- JSON Patch
- JSON Merge Patch
