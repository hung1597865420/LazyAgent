type: entity

Các API auth và config được nhắc đến:

- `@nestjs/passport`
- `@nestjs/jwt`
- session-based auth
- `@nestjs/config`
- `ConfigService`
- `.env` files
- `@nestjs/throttler`
- `@SkipThrottle()`

Đây là các thực thể cấu hình và xác thực ảnh hưởng trực tiếp đến bảo mật ứng dụng.
