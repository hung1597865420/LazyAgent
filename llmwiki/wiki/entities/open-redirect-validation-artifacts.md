Các artifact cần có khi validate:

- minimal URL dẫn tới external domain
- full address bar capture
- bằng chứng bypass regex/allowlist
- chứng minh only first hop validated
- OAuth/SAML code hoặc RelayState delivery tới endpoint attacker-controlled
