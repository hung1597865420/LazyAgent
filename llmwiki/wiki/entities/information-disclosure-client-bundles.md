type: entity
related: [[Information Disclosure]]

Các client-side artifact được nhắc đến:

- source maps (`.map`)
- webpack/Vite maps
- `__NEXT_DATA__`
- static JSON
- `NEXT_PUBLIC_` variables
- `VITE_` variables
- `REACT_APP_` variables

Chúng có thể lộ original sources, comments, internal logic, IDs, flags, hoặc PII.
