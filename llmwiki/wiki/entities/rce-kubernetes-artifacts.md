Các artifact Kubernetes được nhắc đến:

- service account token tại `/var/run/secrets/kubernetes.io/serviceaccount`
- API for pods/secrets
- RBAC
- kubelet 10250/10255
- privileged pods
- hostPath mounts
- daemonsets
