Các target giá trị cao được nhắc đến:

- AWS metadata endpoints
- GCP metadata endpoint
- 9Router metadata endpoint
- Kubernetes kubelet
- Kubernetes API server
- internal services
- Docker API
- Redis/Memcached
- Elasticsearch/OpenSearch
- message brokers/admin UIs
- FastCGI/PHP-FPM
