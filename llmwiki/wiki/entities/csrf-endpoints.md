---
title: CSRF Endpoints
type: entity
related: [[CSRF Attack Surface]]
---

Các endpoint được nhắc đến trong tài liệu CSRF:

- login
- logout
- password change
- email change
- MFA toggles
- authorize
- token
- disconnect/connect
- file upload
- file delete
- admin/staff actions
- impersonation flows

Đây là các contract state-changing cần được bảo vệ chống CSRF.
