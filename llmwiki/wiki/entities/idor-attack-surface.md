Các bề mặt tham chiếu object được nhắc đến:

- Paths
- Query params
- JSON bodies
- Form-data
- Headers
- Cookies
- JWT claims
- GraphQL arguments
- WebSocket messages
- gRPC messages

Đây là các nơi object reference có thể xuất hiện và cần được kiểm tra binding.
