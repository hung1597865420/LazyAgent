Các bề mặt attack surface được nhắc đến:

- form fields
- query/path/header values
- cookies
- JSON / GraphQL variables
- filenames and file metadata
- email subject/body/template-selector fields
- theme/customization endpoints
- Markdown/WYSIWYG content
- template editor features
- YAML/TOML/JSON values later evaluated through templates
