---
title: Next.js Server Action Protocol
type: entity
related: [[Server Actions Security]]
---

Giao thức Server Actions trong Next.js được nhắc đến qua:

- POST request
- `Next-Action` header
- action IDs
- response streams
- hydration data

Đây là contract thực thi action từ client tới server và là nơi cần kiểm tra authorization.
