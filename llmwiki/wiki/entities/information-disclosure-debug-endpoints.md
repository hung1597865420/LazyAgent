Các debug/dev endpoint được nhắc đến:

- Django DEBUG pages
- Laravel Telescope
- Rails error pages
- Flask/Werkzeug debugger
- ASP.NET customErrors Off
- `/debug/pprof`
- `/actuator`
- `/_profiler`
- custom `/debug` APIs

Đây là các endpoint có thể lộ cấu hình, stack, hoặc dữ liệu nội bộ.
