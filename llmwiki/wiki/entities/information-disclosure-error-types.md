title: Information Disclosure Error Types
type: entity
related: [[Information Disclosure]]
---

Các loại lỗi/exception được nhắc đến:

- SQL/ORM errors
- Stack traces
- Template engine probes
- JSON/XML parser errors

Chúng có thể lộ table/column names, absolute paths, class/method names, framework versions, hoặc internal model names.
