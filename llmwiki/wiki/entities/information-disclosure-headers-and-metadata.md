title: Headers and Response Metadata
type: entity
related: [[Information Disclosure]]
---

Các header và metadata được nhắc đến:

- `Server`
- `X-Powered-By`
- `X-AspNet-Version`
- `X-Request-Id`
- `traceparent`
- `Server-Timing`
- debug headers
- `ETag`
- `If-None-Match`
- `Last-Modified`
- `If-Modified-Since`
- `Accept-Ranges`
- `Range`

Chúng có thể dùng để fingerprint stack hoặc làm caching oracle.
