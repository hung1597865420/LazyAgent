title: BFLA High-Value Actions Entity
type: entity
related: [[BFLA High-Value Actions]]
---

Các action giá trị cao được nhắc đến:

- role/permission changes
- impersonation/sudo
- invite/accept into orgs
- approve/void/refund/credit issuance
- price/plan overrides
- export/report generation
- data deletion
- account suspension/reactivation
- feature flag toggles
- quota/grant adjustments
- license/seat changes
- 2FA reset
- email/phone verification overrides

Đây là các operation cần được bảo vệ chặt chẽ ở service boundary.
