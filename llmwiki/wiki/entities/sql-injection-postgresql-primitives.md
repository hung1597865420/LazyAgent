Các primitive PostgreSQL được nhắc đến:

- `version()`
- `current_user`
- `current_database()`
- `xpath()` errors
- `COPY (program ...)`
- dblink
- foreign data wrappers
- `pg_sleep(n)`
- `COPY table TO/FROM '/path'`
- `lo_import`
- `lo_export`
- JSON/JSONB operators
