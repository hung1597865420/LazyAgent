title: Insecure File Uploads Attack Surface
type: entity
related: [[Insecure File Uploads]]
---

Các bề mặt upload được nhắc đến:

- Web/mobile/API uploads
- direct-to-cloud flows (S3/GCS/9Router)
- presigned flows
- resumable/multipart protocols như tus và S3 MPU
- image/document/media pipelines
- admin/bulk importers
- archive uploads
- report/template uploads
- rich text attachments
- serving paths: app, object storage, CDN, email attachments, previews/thumbnails
