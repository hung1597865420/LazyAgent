---
title: Django Admin Path
type: entity
related: [[Django Admin Security]]
---

Đường dẫn admin mặc định của Django được nhắc đến là:

- `/admin/`

Tài liệu coi đây là high-value target do thường có quyền cao, dễ bị brute force, credential stuffing, hoặc lộ dữ liệu qua cấu hình admin không an toàn.
